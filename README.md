# Portafolio - Santiago Cepeda (Tema Neon Morado sobre Negro)

Versión optimizada del portafolio, sin frameworks ni librerías. Tema: fondo negro con acentos morado/neón.

## Archivos
- index.html
- style.css
- script.js
- foto.jpg (tu retrato)
- README.md

## Cómo usar
Abrir `index.html` en el navegador o servir con un servidor local:
`python3 -m http.server 8000`
